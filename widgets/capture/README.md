# Capture

Quickly capture thoughts and tasks without leaving your workflow.

## What it does

Capture is a lightweight inbox widget for the notch.

- type into the input and press `Return` to add a new item
- click a row or checkbox to mark an item complete
- delete items with the trailing trash action
- items persist through the widget's LocalStorage

## Notes

- The widget stores its list under LocalStorage, so it keeps state between app
  launches.
- It is bundled with NotchApp and seeded into the default Home layout for fresh
  installs.
